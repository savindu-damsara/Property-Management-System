# 🌿 GreenLease — Property & Tenant Management System

A full-stack mobile application for managing properties, leases, billing, maintenance requests, and appointments between **property owners** and **tenants**.

---

## 📱 Overview

GreenLease is a cross-platform mobile app built with **React Native (Expo)** and a **Node.js/Express REST API** backed by **MongoDB**. It provides two user roles:

- **Owner** — List and manage properties, handle lease agreements, track payments, respond to maintenance requests, post notices, and manage appointments.
- **Tenant** — Browse available properties, request leases, pay bills, submit maintenance requests, schedule appointments, and view notices.
